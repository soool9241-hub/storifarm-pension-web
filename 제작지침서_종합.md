## 1. pansun_maker 프로젝트 개요

### 1.1 한 줄 정의
> **펜션 사업주**를 대상으로, **OTA 수수료 절감 + 자체 예약률 50% 달성**을 약속하며 홈페이지 제작→유지보수→아카데미→파트너까지 6단계로 태우는 **풀퍼널 SaaS**.

### 1.2 정체성
- **판매 주체**: 광고대행사가 아닌 **"동업종 펜션사업주(임솔, 달팽이아지트 7년 운영)"** 포지셔닝.
- **차별화 축**: 실전 운영자의 시스템 복제 → 신뢰 기반 계약 → 단순 제작이 아닌 **운영 체계 이식**.
- **핵심 약속(Promise)**: OTA 수수료 연간 180~360만원 절감, 자체 예약 30~50%, 납품기간 14~21일.

## 3. 6단계 자동매출 퍼널 (STAGE 1~6)

### STAGE 1 — 진단 (Lead Magnet)
- **경로**: `/diagnostic`
- **산출물**: 20문항 자가진단 → 점수화 → 즉시 리포트 슬러그 발급 + SMS 발송
- **테이블**: `leads`, `diagnostics`
- **UI 원형**: `펜션 사전 진달 앱/pension-diagnostic.jsx`
- **KPI**: 진단 완료율, SMS 수신 동의율

### STAGE 2 — 리포트 (Trust Builder)
- **경로**: `/report/[slug]`
- **기능**: 맞춤 진단 결과 + 개선 포인트 + 제작 서비스 제안 CTA
- **테이블**: `reports`
- **SMS 트리거**: 리포트 열람 시 내부 알림

### STAGE 3 — 제작 (Core Offer)
- **경로**: `/pricing` → `/contact`
- **3티어 요금제**: 라이트 / 스탠다드 / 프리미엄
- **제작기간**: 14~21일 납품
- **테이블**: `consultations`, `contracts`, `packages`
- **차별화**: OTA 수수료 절감 근거 데이터 제시, 달팽이아지트 실사례 공개

### STAGE 4 — 유지보수 (Recurring Revenue)
- **경로**: `/maintenance`
- **모델**: 월 5만원 구독 (도메인/호스팅/콘텐츠 업데이트/문의응대)
- **테이블**: `subscriptions`
- **연계**: `dalpaengi_sms` 40종 SMS 템플릿 재사용

### STAGE 5 — 아카데미 (Knowledge Product)
- **경로**: `/academy`
- **상품**: 온라인 / 원데이 / 풀코스 운영 교육
- **테이블**: `enrollments`
- **참조**: `darr_membership/퍼널구축_아카데미_4주코스.md`

### STAGE 6 — 파트너 (Scale Layer)
- **경로**: `/partners`
- **모델**: 1인 1지역 독점 라이선스(레퍼럴+매출배분)
- **테이블**: `partners`
- **참조**: `tree/referral-tree-app`

## 4. 데이터베이스 스키마 (Supabase, 10 테이블)

| 테이블 | 용도 | 핵심 컬럼 |
|---|---|---|
| `leads` | 모든 진입 리드 원장 | id, phone, name, region, source, created_at |
| `diagnostics` | 20문항 응답/점수 | lead_id, answers(jsonb), score, slug |
| `reports` | 생성된 리포트 | diagnostic_id, slug(public), content(jsonb), viewed_at |
| `consultations` | 상담 신청 | lead_id, package_interest, schedule, status |
| `contracts` | 제작 계약 | lead_id, package_id, amount, deliver_due, status |
| `packages` | 3티어 요금제 마스터 | tier, price, features(jsonb) |
| `subscriptions` | 월 유지보수 | contract_id, monthly_fee, status, next_billing |
| `enrollments` | 아카데미 수강 | lead_id, course_type, price, started_at |
| `partners` | 지역 파트너 | lead_id, region(unique), tier, commission_rate |
| `sms_logs` | Solapi 발송 이력 | phone, template_key, status, sent_at |

**RLS 원칙**: 관리자만 쓰기, 공개 슬러그만 익명 읽기, 본인 리드만 읽기.
